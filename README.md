Remaining setup:

      REPLACE THE VALUES BELOW WITH YOUR ACTUAL GOOGLE FORM DETAILS:

      GOOGLE_FORM_ID   — found in your form URL:
                         https://docs.google.com/forms/d/GOOGLE_FORM_ID/edit

      entry.XXXXXXXXX  — the entry IDs for each field. To find them:
                         1. Open your form
                         2. Click the 3-dot menu > Get pre-filled link
                         3. Fill dummy data in each field, click Get Link
                         4. The URL will contain entry.1234567890=dummy for each field
                         5. Copy each entry ID into the matching variable below
